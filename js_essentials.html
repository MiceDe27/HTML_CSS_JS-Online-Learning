Introduction

JavaScript is a powerful programming language used to create dynamic and interactive effects on websites.

Variables

Variables store data. Use let, const, or var to declare them.

Data Types

JavaScript has different data types like String, Number, Boolean, Object, Array, null, and undefined.

Functions

Functions are blocks of code that perform a task when called. Example: function greet() { alert("Hi!"); }

Loops

Loops help you repeat code. Examples include for, while, and do...while.

All Definitions of JavaScript

  • JavaScript: A programming language for the web
  • Variable: A container for data values
  • Data Type: The type of value (e.g., number, string)
  • Function: A block of code for a task
  • Loop: Repeats a block of code
  • Condition: Logic that checks true or false
  • Event: An action like click, keypress
  • Object: A collection of key-value pairs
  • Array: An ordered list of values
  • DOM: Document Object Model – structure of HTML via JS
