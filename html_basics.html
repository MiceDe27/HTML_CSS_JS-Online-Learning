Introduction

HTML stands for HyperText Markup Language. It is the foundation of every website you visit and is used to create the structure of web pages.

Elements

HTML elements are the building blocks of HTML pages. They include tags like <p>, <h1>, <a>, and many more.

Tags

Tags are keywords enclosed in angle brackets like <html>. They come in pairs — an opening tag and a closing tag — and define the content type.

Table

Tables are used to display data in rows and columns using <table>, <tr>, <th>, and <td>.

List

There are two main types of lists in HTML: unordered lists <ul> and ordered lists <ol>, both using <li> for each item.

All Definitions of HTML

  • HTML: HyperText Markup Language
  • HyperText: Text with clickable links
  • Markup Language: Uses tags to format content
  • Element: Full HTML structure like <p>Text</p>
  • Tag: Command like <h1>, <img>, <a>
  • Attribute: Extra info inside tags, e.g., src, href
  • Content: The text/data inside an element
  • Document: Full HTML page structure
  • Heading: Titles using <h1> to <h6>
  • Paragraph: <p> block of text
  • Anchor: <a> for links
  • Image: <img> for images
  • List: <ul>, <ol>, and <li>
  • Table: <table> with rows and columns
  • Form: Used to get user input
  • Comment: <!-- Notes -->
  • DOCTYPE: Declares HTML version
  • Semantic HTML: Tags that describe meaning
  • HTML5: Latest version with audio, video, canvas
