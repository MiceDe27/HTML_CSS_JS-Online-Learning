Introduction

CSS stands for Cascading Style Sheets. It describes how HTML elements are displayed on the screen, paper, or in other media.

Selectors

CSS selectors are used to "select" the HTML elements you want to style. Examples include element, #id, .class.

Properties

CSS properties define the style to apply, such as color, font-size, margin, padding, etc.

Box Model

Every HTML element is a box made up of: Content, Padding, Border, and Margin.

Layout

CSS layout techniques include flexbox, grid, float, and position to organize content visually.

All Definitions of CSS

  • CSS: Cascading Style Sheets
  • Selector: Targets HTML elements to style
  • Property: The style you want to apply (e.g., color)
  • Value: The value assigned to the property
  • Declaration: A property-value pair (e.g., color: red;)
  • Rule: A full CSS instruction (selector + declarations)
  • Box Model: Content + Padding + Border + Margin
  • Responsive Design: Design that adapts to screen size
  • Flexbox: One-dimensional layout control
  • Grid: Two-dimensional layout system
  • Media Query: CSS rules for specific screen sizes
  • ID Selector: Targets element with specific ID
  • Class Selector: Targets elements with same class
  • Inline CSS: CSS written directly in an element
  • Internal CSS: CSS written in <style> tags
  • External CSS: CSS in a separate .css file
  • Position: Controls element placement (static, relative, absolute, fixed)
  • Z-index: Controls stacking order
